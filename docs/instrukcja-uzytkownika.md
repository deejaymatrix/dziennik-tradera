# Instrukcja użytkownika

> **Status: szkic wstępny (Kamień 0).** Aplikacja nie ma jeszcze żadnego ukończonego modułu produktowego, więc ta instrukcja jest na razie pusta z zamiarem. Zostanie napisana rozdział po rozdziale wraz z powstawaniem kolejnych kamieni (logowanie i onboarding w Kamieniu 1-2, moduły dziennika w Kamieniach 3-5, instalacja/aktualizacja w Kamieniu 6).

6. Backup i przywracanie danych — po Kamieniu 4.
7. Aktualizacja aplikacji (desktop i web/PWA) — po Kamieniu 6.
8. Zapraszanie kolejnych osób (dla właściciela) — po Kamieniu 2/etap II.
9. Eksport i usunięcie danych — po Kamieniu 4.
